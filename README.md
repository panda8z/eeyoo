## 介绍

本项目做为文章微服务的一部分,由 server,web后台,web前端,FlutterApp构成.
提供账号注册登陆和管理,文章发布管理等功能.

Server端功能:

- Mysql数据库
- 日志分割
- 跨域
- 数据验证
- 简易配置
- gin 路由
- jwt

## 软件架构

典型的前后端分离架构, server 端代码用 MVC 组织.
web后端代码使用 AntDesign Pro 组织.
web前端代码使用 ReactJs + GatsbyJs 组织.
